### [sample_data.csv](sample_data.csv)
**Sample CSV File**

A ready-to-use example file with 10 months of realistic subscription data:
- 20 customers
- Mix of active, cancelled, paused subscriptions
- Upgrades and downgrades
- New customers and churn

**Use this to test SubAudit before uploading your own data.**

**What this sample will show:**
- ✅ All 5 metric blocks (Revenue, Growth, Retention, Health, Cohort)
- ✅ 12-month forecast (10 months of data = all 3 scenarios)
- ✅ Cohort retention heatmap
- ✅ Data quality report

**To use:**
1. Download [sample_data.csv](sample_data.csv)
2. Go to [SubAudit](https://subaudit.streamlit.app)
3. Upload the file
4. Follow the mapping and cleaning steps
5. View your dashboard!
